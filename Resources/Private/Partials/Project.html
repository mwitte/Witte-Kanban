W:Kanban

This project stated from a personal idea to learn something about TYPO3 Flow.

Features
  • Create a Kanban board
  • Create and delete columns
  • Create, edit, delete, archive Tickets
  • Switch tickets between columns
  • Responsive layout
Planned features
  • Create and delete sub columns
  • Multi-language support
  • Swim lanes
  • Drag and drop for tickets
  • Fileupload for tickets
  • Detailed statistics
